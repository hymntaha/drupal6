Publish new post or post update notifications to Ping.fm. You can share you're blog with 30+ social services.

== Description ==

Publish 2 PingFM allows you to automatically publish your posts to the Ping.fm service. This plugins allows you to :
* select for witch category you want to publish your posts to ping.fm
* select a different publishing template for each category
* select, for each category, if you want to publish posts updates or not
* select, for each category, a different template for post updates than for initial publishing
* shorten posts URL using bit.ly shortening service

== Installation ==

1. Upload `publish2pingfm` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it, all done!

== Frequently Asked Questions ==

= Do I have to use Bit.ly? =

For the moment yes, a version with the option not to shorten link will be available soon

= What can I put in the templates =

In the templates you can put any text. In addition, the following tags will be replaced :
* [title] will be replaced by the post title
* [url] will be replaced by the post long URL
* [short_url] will be replaced by the post shortened URL
